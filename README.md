# scrap_revuedepresse

## Requirements

- python-lxml
- python-urllib3
- python-beautifulsoup4
- python-requests
- python-selenium
- geckodriver
- firefox
- opencv
- stapler
- python-pandas
- hdf5
- imagemagick (convert pdf to image)
- ghostscript (imagemagick pdf support)

## Installation

```
python setup.py install
```

## Usage

```
scrap_revuedepresse
```

## Help

```
scrap_revuedepresse -h
```
